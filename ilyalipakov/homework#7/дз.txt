1) необходимо разработать функцию, которая получив id страницы рендерит карточки 
пользователей, в HTML. Процесс рендеринга страницы из JSON необходимо реализовать 
отдельной функцией.
Ссылка для запросов https://reqres.in/api/users?page=1

2) реализовать процесс, который будет вызывать функцию из первого 
пункта при скролинге страницы. Когда до конца страницы вам осталось 
100 или меньше пикселей необходимо загрузить новую страницу данных. 
Таким образом вы добьетесь бесконечной ленты, как в интсаграме и подобных сервисах. 

3) * реализовать кнопку удаления у каждой карточки пользователя, 
которая отправляет запрос на удаление на сервер и при успешном ответе
 удаляет элемент из DOM

4) * реализовать форму по добавлению карточки пользователя.
 Данные из формы отправляются на сервер и при положительном 
 ответе элемент должен рендериться в DOM 

ВНИМАНИЕ!! сервис reqres не сохраняет состояние и процессы
удаления/добавления лишь эмулируют действия,
но не изменяют данные на сервере.
